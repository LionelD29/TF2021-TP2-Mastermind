Rules

The game rules are simple:

The player has to find a secret combination of 4 colors in a given number of attempts.

At each attempt, the player receives hints concerning the colors and their position

The color is correct
The color is used but at a wrong position
The color is not used in the solution

Game mode

In singleplayer mode, the secret code is randomly generated by the game

In multiplayer mode, one player chooses a secret code and the other has to guess it.

Level of difficulty

In the settings, you can set the difficulty of the game by choosing the number of attempts as well as the number of available colors. You can choose between 1 and 15 attempts, and 4 to 7 available colors
